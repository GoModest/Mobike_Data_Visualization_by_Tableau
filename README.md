# Mobike_Data_Visualization_by_Tableau
使用Tableau对摩拜单车数据进行可视化分析

文件列表：
- Mobike data.zip: 摩拜单车原始数据
- 把摩拜单车数据进行转换：用来把原始数据中的track提取后排序的Notebook
- 摩拜单车.pdf：设计思路总结。
- 摩拜单车.twb：第一版可视化分析文件。
- 摩拜单车2.0.twb:第二版可视化分析文件。
